# IsskinUI

**IsskinUI** is a React component library built to empower all Isskin projects and services with a unified, accessible, and reusable set of UI elements.

Crafted to reflect Isskin's brand identity and design principles, IsskinUI helps teams deliver consistent, seamless user experiences across all products.

## 🚀 Installation

```bash
npm install isskinui
```

## 📦 Usage

```tsx
import { Button } from "isskinui";

function App() {
  return <Button label="Click me" />;
}
```

⚠️ Make sure your app is wrapped in any required providers (e.g., theme or localization) as needed by your components.
